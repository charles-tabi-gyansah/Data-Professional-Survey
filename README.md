# Data Professional Survey Analysis with Power BI

## Project Overview
This project analyzes a Data Professional Survey using Power BI. The workflow includes data cleaning and transformation using Power Query, followed by data modeling and interactive visualizations to uncover insights about data professionalsâ€™ roles, salaries, tools, and job satisfaction.

This project demonstrates the ability to handle end-to-end business intelligence tasks within Power BI.



## Tools & Technologies
- Power BI Desktop
- Power Query (ETL: Extract, Transform, Load)
- DAX (Basic Measures)
- Interactive Dashboards & Visualizations



## Dataset Description
The dataset is based on a survey of data professionals and includes information such as:
- Job roles and career titles
- Salary ranges
- Programming languages used
- Industry and location
- Work-life balance and job satisfaction metrics



## Data Cleaning & Transformation (Power Query)
- Removed duplicate and irrelevant records
- Cleaned and standardized column names
- Transformed data types for accurate analysis
- Handled null and missing values
- Created new columns for analysis readiness



## Analysis Performed
- Salary distribution across different data roles
- Popular programming languages among data professionals
- Job satisfaction and work-life balance trends
- Comparison of roles by experience and compensation



## Visualizations & Dashboard
- Bar charts for role and salary comparisons
- Donut and column charts for tool usage

- Interactive filters and slicers for dynamic exploration
